Privacy Policy

Daily Log Chrome Extension

The short version

Daily Log does not collect, transmit, sell, or share any personal data. Everything you enter stays inside your browser on your device.

What data the extension handles

Daily Log stores the following locally in your browser, using the browser's localStorage API:

  • The list of goals you create (names and target minutes)
  • The list of note prompts you create (labels)
  • Today's draft values (slider positions, text in note fields)
  • Per-day history of how many minutes you logged and how many goals you hit

This data never leaves your device. The extension makes no network requests. There is no server, no account, no telemetry, and no analytics.

What the extension does NOT do

  • No accounts or sign-in
  • No cloud sync
  • No tracking pixels or analytics services
  • No ads
  • No reading or modifying any web page you visit
  • No access to your browsing history, bookmarks, cookies, or other extensions
  • No access to your files except the single markdown or JSON file you explicitly create via Save today's log or Export backup

Permissions the extension requests

The extension's manifest.json declares only the bare minimum permissions:

  • action — to display a toolbar icon you can click
  • background — to handle the click and open the Daily Log tab

It does NOT request tabs, host permissions, storage (it uses localStorage instead, which doesn't require a permission declaration), activeTab, cookies, or any other permission category that lets an extension see what you're doing.

File downloads and uploads

Two features create files on your computer, and both require an explicit click each time:

  • Save today's log downloads a single .md file to your Downloads folder. Nothing is uploaded.
  • Export backup downloads a single .json file to your Downloads folder. Nothing is uploaded.
  • Import backup opens your system file picker so you can choose a .json file to restore. The file is read in your browser only. Nothing is uploaded.

Children's privacy

The extension does not collect any data from any user, so it does not knowingly collect data from children. It is safe for use by people of any age, subject to the Chrome Web Store's own age requirements.

Changes to this policy

If the extension ever changes in a way that affects what data is handled or how, this page will be updated and the version date below will change.
